I was first introduced to anime in grade 6 with Attack on Titan, but did not start watching actively until high school when my friends introduced me to One Piece. Projects

Manga Tracker

As mentioned earlier in my interests, I enjoy reading manga and I used to read them from websites online. I found that checking the website to see if there was a new chapter for all of the manga that I was reading was very tedious and was often a waste of time if there were no new chapters. Their sites were usually filled with lots of ads and popups which made visiting them very unpleasant. There was also pressure to read the chapters as soon as they were released in order to avoid getting spoiled from other sources such as social media or friends. I decided to make a web scraper using python that would find out if there was a new release for the manga that I read and would send an email with the chapters as attachments. I had set this script to run every hour so I now get chapters almost as soon as they are released and the emails contain just the pages with no ads so it makes for a pleasurable reading experience.
